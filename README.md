### Setting koneksi database
Buka file `database.ini`, kemudian ubah sesuai koneksi database yang ada di local
```
; this is an INI file
[connection_database]
url=jdbc:postgresql://localhost:5432/pbo
username=naufalnibros
password=naufalnibros

```
